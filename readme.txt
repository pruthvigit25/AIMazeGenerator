1. open folder 
2. Right click and click on open in terminal.
3. Windows Powershell command window will pop up.
4. Type the below commands to run the maze files
   > py
   > import run_maze
   > run_maze.maze_algorithms() #runs all the five algorithms
   > run_maze.maze_algorithms(5,10,'dfs') #runs dfs maze file
   > run_maze.maze_algorithms(5,10,'bfs') #runs bfs maze file
   > run_maze.maze_algorithms(5,10,'astar') #runs astar maze file
   > run_maze.maze_algorithms(5,10,'vi') #runs MDP Value Iteration maze file
   > run_maze.maze_algorithms(5,10,'pi') #run MDP Policy Iteration maze file
